brisky
============

Require Global Modules
----

node, npm

How to build
----
```
$ npm install
$ npm run build
```

How to run
----

```
$ export HTTPS_PROXY=[http(s)://[USER]:[PASSWORD]@[PROXY_HOST]:[PROXY_PORT]]
$ export JIRA_USERNAME=[JIRA_USERNAME]
$ export JIRA_PASSWORD=[JIRA_PASSWORD]
$ npm start
```

How to stop
----

```
$ npm stop
```
